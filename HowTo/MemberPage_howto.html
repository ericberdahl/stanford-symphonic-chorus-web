Member Tools

This section contains resources for chorus members and changes every quarter.

Rehearsal Schedule

The following links to the syllabus change every quarter. Be sure to add each quarter’s syllabus files to the performances/qYYYY folder for the quarter.

Spring 2014 Syllabus (PDF)
Spring 2014 Syllabus (Word docx format)

A link here to the online-registration form changes each quarter and can be hidden by commenting it out after the deadline. The online-registration link looks like this:

General information about the registration process does not change much; however, any date mentioned must be updated regularly. Also watch for changes to the registration fee amount and the cost of the scores.

Other links, such as to Joanne Bogart’s Italian diction guide, should be added here, or removed from here, as needed.

Joining the Chorus

Here, only the participation fee amount is likely to need to be updated.

Auditions

The first paragraph here, on scheduling auditions, changes regularly, particularly regarding the dates and times of auditions, and whether the chorus is currently full. Be sure to use the unScramble function to hide any email address from web robots.

The rest of the section is generally stable.

Expectations

Expectations do not generally need to be updated.
