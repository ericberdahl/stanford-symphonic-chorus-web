Obtaining Practice Files for Stanford Summer Chorus 04

The practice music files are, for now, available in Windows Media format and MP3. They're quite large (from .5 to 2 megabytes for a single file for Windows Media and about twice as large for MP3!).

Files for each section are in a separate directory:

Windows Media (preferred for Windows machines)

  • soprano
  • alto
  • tenor
  • bass

MP3 (for Macs, Linux, etc.)

  • soprano
  • alto
  • tenor
  • bass

To hear a file, you might need to copy it to your local disk. Usually you can do this by right-clicking on it in your browser.
